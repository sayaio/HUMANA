Letakkan file service account Firebase di folder ini dengan nama PERSIS:

    firebase-service-account.json

File ini didapat dari:
  Firebase Console > Project humana-5ed56 > Project Settings (ikon gerigi)
  > tab "Service accounts" > tombol "Generate new private key".

JANGAN commit file JSON tersebut ke git (sudah di-ignore lewat .gitignore).
Setelah file diletakkan di sini, restart backend (node index.js).
